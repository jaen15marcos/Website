Easy Problems

Problem 1: Create an n x n matrix of random floats between a and b

{[n;a;b] (n;n)#a+(n*n)?b}[5;0;1.0]

Problem 2: Isolate the ith row or jth column

show random_matrix: {[n;a;b] (n;n)#a+(n*n)?b}[5;0;1.0];
{[i;j] random_matrix[i;j]}[2;0] /for specific cell - row i, col j 
{[j] (flip random_matrix)[j]}[0] /isolate col j
{[j] random_matrix[j]}[0] /isolate row i

Problem 3: Isolate the leading diagonal (top left to bottom right)

show random_matrix: {[n;a;b] (n;n)#a+(n*n)?b}[5;0;1.0];
random_matrix @' til 5

Problem 4: Find the transpose

flip random_matrix

Problem 5: Create identity matrix

{0 1f x =/: x}til 10

Problem 6: Sort each row/column in ascending order

{[x] asc x}each random_matrix /asc rows
flip {asc x}each flip random_matrix /asc cols

Problem 7: Find the index of the max/min element in each row

{max x}each random_matrix /max element by row
{max x}each flip random_matrix /max element by row

Problem 8: Convert the symbols `rix`mat to the symbol `matrix

`$"" sv (string `mat; string `rix)

Problem 9: Given a string x and a character y, how many times does y occur in x

{[x;y] sum x=y}["fhqwhgads";"h"]

Problem 10: Given a string x, is it identical when read forwards and backwards?

{[x] (reverse x)~x}["racecar"]

Problem 11: Given a string x, produce a list of characters which appear more than once in x

{[x] where 1< count each group x}["applause"]

Problem 12: Given strings x and y, do both strings contain the same letters, possibly in a different order?

{[x;y] (asc x)~(asc y)}["teapot";"toptea"]

Problem 13: Given a string x, find all the characters which occur exactly once, in the order they appear.

{[x] where 1=count each group x}["somewhat heterogenous"]

Problem 14: Given strings x and y, is x a rotation of the characters in y?

{[x;y] x in (1 rotate) scan y}["foobar";"barfoo"]

Problem 15: Given a list of strings x, sort the strings by length, ascending.

{x iasc count each x}[("books";"apple";"peanut";"aardvark";"melon";"pie")]

Problem 16: Given a string x, identify the character which occurs most frequently.

x: "abdbbac";first key desc count each group x
(raze/) x where (value count each group x)=(max desc count each group x)

Hard Problems

Problem 1: Armstrong number: An n-digit number equal to the sum of the nth powers of its digits. EX1: 123 is a three-digit number, raise each digit to the third power, and add: 1^3 + 2^3 + 3^3 = 36, which shows that it is not an Armstrong number. EX2: 1634 is Armstrong number as 1^4 + 6^4 + 3^4 + 4^4 = 1634. Get all Armstrong numbers up to 10k.

isArmsNum:{[no] no=sum {[c;n] ("I"$n) xexp c }[count string no]each string no}
isArmsNum 1634 
(2 + til 10000000) where (isArmsNum'[2 + til 10000000])

Problem 2: Each new term in the Fibonacci sequence is generated by adding the previous two terms. By starting with 1 and 2. By considering the terms in the Fibonacci sequence whose values do not exceed four million, find the sum of the even-valued terms.

fibSeq:-1_{x,sum reverse[x]0 1}/[{(4000000)>last x};1 2] 
sum fibSeq where not fibSeq mod 2
fibonnaci: {[N;l] N {x, sum -2#x}/l}[N; 1 2] /N=30
sum fibonnaci where not fibonnaci mod 2

Problem 3: Find the greatest product of five consecutive digits in the 1000-digit number.

number: raze string 1000?1 + 10
number: ((1000 - count[number])_number)
max {prd{"I"$ x}each number x+til 5}each til 1000
